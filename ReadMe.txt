Instructions:Please note the following instructions are based on OSX operating systems.1. Install PHPUnit to local machine via terminal: wget https://phar.phpunit.de/phpunit.phar  chmod +x phpunit.phar  sudo mv phpunit.phar /usr/local/bin/phpunit  phpunit --version PHPUnit 6.2.0 by Sebastian Bergmann and contributors.Alternatively please see: https://phpunit.de/getting-started.html.2. Add 'swm' folder to PHP Document root 3. On terminal go to the swm folder (created above)4. Change directory to 'swm' folder5. Run phpunit test with the following cmdPhpunit ShoppingCartTest.php6. It should display something like thisrwijaya$ phpunit ShoppingCartTest.php PHPUnit 6.2.3 by Sebastian Bergmann and contributors.....                                                                4 / 4 (100%)Time: 488 ms, Memory: 8.00MBOK (4 tests, 13 assertions)
